Задание: создание сайта

При создании сайта были применины следующие теги и атрибуты к ним:

html - тег начала html-кода (и его окончания "/html");

head - "голова" html-кода (предназначен для хранения в нем тега "title");

title - тег заголовка вкладки окна страницы;

body - "тело" html-кода, этот тег наполняет код станицы содержимым. В нем содержатся теги картинок, таблиц, текста;

h1 (h1- h6) - тег заголовка 1-6-го уровней, h1 - самый большой заголовок, h6 - самый маленький заголовок;

br - тег перехода на следующую строку;

P - тег текста в html-коде (атрибуты: выравнивания "align=" по центру "center", по левому краю "left", по правому краю "right", по ширине "justify");

B - этот тег делает шрифт жирным;

I - этот тег делает шрифт курсивом;

U - это тег делает шрифт подчеркнутым;

table - тег таблицы в html-коде (атрибуты tr - начало новой строки, td - начало ячейки столбца);

a href - тег перехода на по ссылке;

А также атрибуты "color=" - цвет шрифта, "size=" - размер шрифта, "bgcolor=" - цвет фона, "border=" - атрибут устанавливающий границу элемента, "cellspacing=" - "расстояние между внешними границами ячейки равно, "cellpadding=" - расстояние между границей ячейки и ее содержимым, "valign=top" - вертикальное выравнивание ячейки по ее верхней, "alt=" - устанавливает альтернативный текст изображения, текст пояснений в html-коде, "rowspan=" устанавливает число ячеек, которые должны быть объединены по вертикали.
